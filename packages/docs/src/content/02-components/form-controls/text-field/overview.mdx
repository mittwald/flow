## Playground

Verwende `<TextField />`, um ein Textfeld anzuzeigen. Das TextField muss immer
um ein `<Label />` erweitert werden. Das Label vermittelt Informationen die für
das Verständnis der Eingabeanforderungen unerlässlich sind. Optional kann auch
eine `<FieldDescription />` ergänzt werden, um zusätzliche Informationen
bereitzustellen.

## States

Das TextField hat 6 verschiedene states: **Default**, **Hover**, **Focused**,
**Disabled**, **Error** und **Error-focused**.

### Disabled

Wenn das TextField disabled ist, zeigt es optisch an, dass es momentan nicht
verfügbar ist. Sobald die Ursache für die Deaktivierung behoben ist, kann das
TextField wieder normal verwendet werden.

### Error and error-focused

Wenn die Benutzereingabe nicht den erwarteten Anforderungen entspricht, wechseln
