Benutze die `<Icon />` Komponente in einem `<Button />` um ein Icon
darzustellen.

### Accessibility

Das enthaltene SVG erhält automatisch das `aria-hidden=true` Attribut. Wenn dein
Button keinen Text enthält, benutze das `aria-label` Attribut um die hinterlegte
Aktion zu beschreiben.
