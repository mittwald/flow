# States

Das Label hat 2 verschiedene States: **Default** und **Disabled**.

<LiveCodeEditor example="states" editorCollapsed />

**Disabled**: Das Label wird häufig zur Beschreibung komplexerer Komponenten
verwendet. Daher übernimmt es automatisch den State, den die höhere Komponente
hat. Ist zum Beispiel ein [TextField](/03-components/form-controls/text-field)
oder [NumberField](/03-components/form-controls/number-field) gerade
**disabled**, nimmt das Label den gleichen State an.
