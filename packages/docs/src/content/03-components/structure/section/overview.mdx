# Kombiniere mit...

## Actions

Die Section unterstützt in der rechten oberen Ecke einen Bereich, der für
Actions reserviert ist. Dort kann ein [Link](/03-components/navigation/link),
[Switch](/03-components/form-controls/switch),
[Button](/03-components/actions/button) oder eine Kombination dieser Components
verwendet werden.

<LiveCodeEditor example="link" editorCollapsed />

<LiveCodeEditor example="switch" editorCollapsed />

Es können maximal zwei Buttons nebeneinander dargestellt werden. Gibt es mehr
als zwei Buttons, sollte nur die Hauptaktion sichtbar sein. Alle anderen Buttons
müssen in ein [ContextMenu](/03-components/actions/context-menu).

<LiveCodeEditor example="context-menu" editorCollapsed />
