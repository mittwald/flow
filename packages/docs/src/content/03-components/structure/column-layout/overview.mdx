# Beispiel

Im Default werden die Spalten folgendermaßen gesetzt:

- s: `[1]`
- m: `[1, 1]`
- l: `[1, 1, 1]`

# Benutzerdefinierte Values

Größere Größen erben von kleineren größen, sofern sie nicht gesetzt sind:

- s: `[1]` (default)
- m: `[2, 1]`
- l: `[2, 1]` (inherited)
