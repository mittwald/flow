## Innerhalb eines Buttons

Um sich bei der Verwendung auf anderen Elementen besser von diesen abzuheben,
hat das CounterBadge eine Border. Diese ist im Default weiß und muss jeweils an
die Hintergrundfarbe angepasst werden.

### Accessibility

Da das CounterBadge selbst nicht accessible ist, muss der Wert über das
darunterliegende Element vermittelt werden, in diesem Beispiel erhält der Button
ein Aria Label, das die Zahl des Counters enthält.
