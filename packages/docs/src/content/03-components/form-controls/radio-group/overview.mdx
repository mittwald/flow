## Container Breakpoint Size

Die segmented RadioGroup springt in die kompakte Variante um, sobald der
Container kleiner als gesetzte Breakpoint ist. Im default passiert das bei
550px, was der `containerBreakpointSize="m"` entspricht. Über dieses Property
kann die Größe mit Werten zwischen `xs` und `xl` überschrieben werden.
