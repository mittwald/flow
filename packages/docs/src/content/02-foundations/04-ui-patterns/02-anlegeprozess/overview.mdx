# List

## Leerer Inhalt

Wenn sich noch kein Element (ListItem) in der List befindet, wird eine
IllustratedMessage angezeigt. Ein Klick auf den Button öffnet in den meisten
Fällen ein Modal – bei einfachen Anlegeprozessen in der üblichen
Modal-Darstellung, bei umfangreicheren Prozessen in der OffCanvas-Variant.

## Mit Inhalt

In einer List oben rechts können weitere Elemente/ListItems angelegt/gebucht
werden. Auch hier öffnet ein Klick auf den Button entweder ein Anlege-Modal für
kleinere Prozesse oder ein Modal in der OffCanvas-Variante für größere
Anlege-/Buchungsprozesse.
