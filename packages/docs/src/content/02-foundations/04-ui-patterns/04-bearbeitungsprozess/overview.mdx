# Section-Inhalte bearbeiten

Durch die generelle Strukturierung von Inhalten mit einer Section ist es
möglich, im oberen Bereich der Section einen "Bearbeiten"-Button zu platzieren.
Dieser öffnet je nach Länge des Bearbeitungsformulars ein Standard-Modal oder
ein Modal in der OffCanvas-Variante.
