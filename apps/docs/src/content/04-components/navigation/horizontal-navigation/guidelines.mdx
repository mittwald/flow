# Grundlagen

## Best practices

Achte bei der Verwendung einer HorizontalNavigation darauf, dass ...

- die Navigationspunkte klar beschriftet und in sinnvoller Reihenfolge
  angeordnet sind — die vorderen Links bleiben bei wenig Platz am längsten
  sichtbar.
- ausschließlich Links enthalten sind. Für Aktionen oder Buttons eignet sich die
  [HeaderNavigation](/04-components/navigation/header-navigation/overview).
- die Navigation nicht überladen wird und nur relevante Bereiche enthält.

## Verwendung

Verwende eine HorizontalNavigation, um ...

- zwischen gleichrangigen Unterbereichen einer Seite oder eines Objekts zu
  navigieren.
- eine platzsparende, responsive Navigation bereitzustellen, die bei wenig Platz
  automatisch in ein ContextMenu zusammenklappt.

Verwende stattdessen [Tabs](/04-components/navigation/tabs/overview), wenn
Inhalte innerhalb derselben Seite umgeschaltet werden, ohne dass eine Navigation
zu einer anderen URL stattfindet.
