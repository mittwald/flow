# States

Das Label hat 2 verschiedene States: **Default** und **Disabled**.

<LiveCodeEditor example="states" editorCollapsed row />

**Disabled:** Das Label wird häufig zur Beschreibung komplexerer Components
verwendet. Daher übernimmt es automatisch den State, den die höhere Component
hat. Ist zum Beispiel ein
[TextField](/04-components/form-controls/text-field/overview) oder
[NumberField](/04-components/form-controls/number-field/overview) gerade
**disabled**, nimmt das Label den gleichen State an.
