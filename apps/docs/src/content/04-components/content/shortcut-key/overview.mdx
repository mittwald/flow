# Kombiniere mit …

## SearchField

Wenn eine Suche durch einen Keyboard-Shortcut fokussiert werden kann, kann der
ShortcutKey mit dem [SearchField](Link) kombiniert werden, um dem User das
verfügbare Tastaturkürzel sichtbar zu machen.

<LiveCodeEditor editorCollapsed example="search-field" />

## ContextMenu

In [ContextMenus](Link) werden häufig Aktionen angezeigt, die zusätzlich per
Keyboard-Shortcut ausgelöst werden können. In diesem Fall kann der ShortcutKey
mit dem jeweiligen Menu-Item kombiniert werden.
