Die Komponente unterstützt sowohl `children` als auch ein Array von `keys`. Die
einzelnen keys werden als Strings übergeben. Wenn dabei die Werte "mod", "alt"
oder "shift" verwendet werden, wird automatisch das passende Tastensymbol
angezeigt – abhängig davon, ob der Nutzer macOS oder Windows verwendet. Dadurch
werden die jeweiligen Tastenbezeichnungen plattformspezifisch korrekt
dargestellt, ohne dass zusätzliche Logik implementiert werden muss.
