# Mehrere Dateien

Über die Property `multiple` wird das gleichzeitige Hochladen mehrerer Dateien
ermöglicht. Beim erneuten Hochladen werden bestehende Dateien ersetzt. Wenn
Dateien nacheinander hochgeladen werden sollen, kombiniere die FileDropZone mit
der [FileCardList](/04-components/upload/file-card-list/overview).

<LiveCodeEditor example="multiple" editorCollapsed />

---

# Kombinieren mit...

## FileCardList

Wenn User Dateien nach dem Hochladen hinzufügen und auch wieder entfernen
sollen, kombiniere die FileDropZone mit der FileCardList. Die
[FileCardList](/04-components/upload/file-card-list/overview) zeigt die
zwischengespeicherten Dateien.
