# Grundlagen

## Best practices

Achte bei der Verwendung einer LayoutCard darauf, dass ...

- der Content innerhalb sinnvoll strukturiert ist. Dies kann beispielsweise
  durch die Components [Section](/04-components/structure/section/overview) und
  [ColumnLayout](/04-components/structure/column-layout/overview) erfolgen.
- sie in wWeiß eingefärbt ist. Wenn Inhalte besonders hervorgehoben werden
  sollen, kann sie mit der
  [AccentBox](/04-components/structure/accent-box/overview) kombiniert werden.

## Verwendung

Verwende eine LayoutCard, um ...

- einen oder mehrere Content-Bereiche auf einer Seite zu erzeugen.
- eine Kacheloptik auf einem Dashboard darzustellen.
- eine äußere Hülle für ein Navigationsmenü zu gestalten.
