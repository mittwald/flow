# Grundlagen

Wir haben die Align Komponente entwickelt, um häufig wiederkehrende
Kombinationen von Komponenten konsistent darzustellen. Sie sorgt für visuelle
und funktionale Einheitlichkeit in unseren Oberflächen – ohne, dass für jede
dieser Kombinationen eine eigene Komponente entwickelt werden muss.

## Best practices

Achte bei der Verwendung von Align darauf, dass ...

- nur die im Overview genannten Komponenten kombiniert werden können.
- es nicht als generisches Layout-Tool dient. Für eine freie Anordnung ist
  [Flex](/04-components/structure/flex/overview) vorgesehen.
- neben einer korrekten Anordnung auch visuelle Anpassungen (z. B. das Setzen
  von Farben) vorgenommen werden können.

## Verwendung

Verwende Align, um ...

- vorab definierte Komponenten (z. B.
  [Button](/04-components/actions/button/overview) und
  [TextField](/04-components/form-controls/text-field/overview)) sinnvoll
  anzuordnen.
- wiederkehrende UI-Patterns konsistent umzusetzen.
