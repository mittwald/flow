# Grundlagen

## Best Practices

Achte bei der Verwendung eines CodeEditors darauf, dass die passende Sprache im
`language` Property gesetzt ist, damit korrektes Syntax-Highlighting möglich
ist.

## Verwendung

Verwende einen CodeEditor, um Code direkt innerhalb der Oberfläche zu schreiben
oder zu bearbeiten. Für die reine Anzeige von Code ist der
[CodeBlock](/04-components/content/code-block/overview) vorgesehen.
