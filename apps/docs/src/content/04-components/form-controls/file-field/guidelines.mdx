# Grundlagen

Das FileField ermöglicht den manuellen Upload einer Datei über eine kompakte
[Button](/04-components/actions/button/overview)-Darstellung. Wenn genügend
Platz vorhanden ist und Drag-and-Drop gewünscht wird, sollte stattdessen die
[FileDropZone](/04-components/upload/file-drop-zone/overview) eingesetzt werden.

## Best practices

Achte bei der Verwendung eines FileFields darauf, dass ...

- ein aussagekräftiges [Label](/04-components/content/label/overview) vorhanden
  ist, falls das FileField sonst nicht eindeutig verständlich wäre.
- die zulässigen Dateitypen und ggf. Größenbeschränkungen deutlich kommuniziert
  werden.
- hochgeladene Dateien übersichtlich dargestellt werden – z. B. als
  [FileCard](/04-components/upload/file-card/overview) oder
  [FileCardList](/04-components/upload/file-card-list/overview).
- [Fehlermeldungen](/02-foundations/03-content-guidelines/02-fehlermeldungen)
  klar und hilfreich formuliert sind.

## Verwendung

Verwende ein FileField, um einen kompakten
Upload-[Button](/04-components/actions/button/overview) in der
Benutzeroberfläche bereitzustellen.
