# Grundlagen

Die FileCardList ergänzt oft die [FileDropZone](/03-components/upload/file-drop-zone) oder das [FileField](/03-components/form-controls/file-field) als visuelle Rückmeldung mehrere Dateien nach dem Upload.


## Best practices

Achte bei der Verwendung einer FileCardList darauf, dass...

- hochgeladene Dateien sofort sichtbar werden, um dem User eine direkte Rückmeldung zu geben.
- relevante Informationen wie Dateiname, Dateigröße und Dateityp klar erkennbar sind.
- wichtige Interaktionen als Aktionen angeboten werden z.B. das Entfernen der Datei.
- für einzelne Dateien die Component [FileCard](/03-components/upload/file-card) verwendet werden kann.



## Verwendung

Verwende eine FileCardList, um...

- dem User bei erfolgreichen Uploads sofort visuelles Feedback zu geben.
- hochgeladene Dateien (Bild, Video oder PDF) darzustellen.
