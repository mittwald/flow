# Grundlagen

Kombiniere die FileDropZone mit der
[FileCard](/03-components/upload/file-card/overview), um einzelne hochgeladene
Dateien darzustellen, oder mit der
[FileCardList](/03-components/upload/file-card-list/overview), um mehrere
hochgeladene Dateien darzustellen.

## Best practices

Achte bei der Verwendung einer FileDropZone darauf, dass ...

- immer eine alternative Upload-Möglichkeit über einen
  [Button](/03-components/actions/button/overview) vorhanden ist.
- die zulässigen Dateitypen und ggf. Größenbeschränkungen deutlich kommuniziert
  werden.
- [Fehlermeldungen](https://mittwald.github.io/flow/02-foundations/03-content-guidelines/02-fehlermeldungen)
  klar und hilfreich formuliert sind.
- die hochgeladenen Dateien übersichtlich durch die
  [FileCard](/03-components/upload/file-card/overview) oder
  [FileCardList](/03-components/upload/file-card-list/overview) dargestellt
  werden.

## Verwendung

Verwende eine FileDropZone, um eine oder mehrere Dateien gleichzeitig hochladen
zu können.
