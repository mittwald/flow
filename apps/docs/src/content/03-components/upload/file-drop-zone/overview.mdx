# Mehrere Dateien

Über die Property `multiple` wird das gleichzeitige Hochladen mehrerer Dateien ermöglicht. Beim erneuten Hochladen werden bestehende Dateien standardmäßig ersetzt. Wenn Dateien nacheinander hochgeladen werden sollen, kombiniere die FileDropZone mit der [FileCardList](/03-components/upload/file-card-list/overview).

<LiveCodeEditor example="multiple" editorCollapsed />

--- 

# Kombinieren mit... 

## FileCardList

Wenn User Dateien nach dem Hochladen hinzufügen und auch wieder entfernen sollen, kombiniere die FileDropZone mit der FileCardList. Die [FileCardList](/03-components/upload/file-card-list/overview) zeigt die im State zwischengespeicherten Dateien.
