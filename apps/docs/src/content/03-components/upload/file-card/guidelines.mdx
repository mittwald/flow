# Grundlagen

Die FileCard ergänzt oft die
[FileDropZone](/03-components/upload/file-drop-zone) oder das
[FileField](/03-components/form-controls/file-field) als visuelle Rückmeldung
nach dem Upload.

## Best practices

Achte bei der Verwendung einer FileCard darauf, dass ...

- hochgeladene Dateien sofort sichtbar werden, um dem User eine direkte
  Rückmeldung zu geben.
- relevante Informationen wie Dateiname, Dateigröße und Dateityp klar erkennbar
  sind.
- mehrere Dateien übersichtlich mit der Component
  [FileCardList](/03-components/upload/file-card-list) dargestellt werden.
- wichtige Interaktionen als Aktionen angeboten werden z. B. das Entfernen der
  Datei.

## Verwendung

Verwende eine FileCard, um ...

- dem User bei erfolgreichen Uploads sofort visuelles Feedback zu geben.
- hochgeladene Dateien (Bild, Video oder PDF) darzustellen.
