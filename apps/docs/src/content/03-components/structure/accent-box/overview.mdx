# Playground

In der AccentBox können Inhalte platziert werden, die sich vom übrigen Content
abheben sollen. Die AccentBox sorgt hierbei für die Hintergrundfarbe und das
Padding. Die Inhalte darin können über Struktur-Komponenten wie `Section` oder
`Flex` angeordnet werden.
