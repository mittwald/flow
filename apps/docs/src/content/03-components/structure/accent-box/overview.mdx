# Playground

In der `<AccentBox />` lassen sich Inhalte platzieren, die sich bewusst vom restlichen Content abheben sollen. Die Component sorgt dabei für die Hintergrundfarbe und das
Padding. Die Inhalte innerhalb der Box können mithilfe von Struktur-Components wie [Section](/03-components/structure/section) oder [Flex](/03-components/structure/flex) angeordnet werden.

# Color

Je nach Anwendungsfall stehen für die AccentBox die Colors **Blue, Green, Neutral** und **Gradient** zur Verfügung. Blue ist dabei als Standard definiert.

## Green

Bei Verwendung der Color Green erscheinen enthaltene Components automatisch in der Dark-Variante (siehe [Light und Dark Color](/02-foundations/01-design/02-colors)) dargestellt, um die nötigen Kontraste sicherzustellen.
