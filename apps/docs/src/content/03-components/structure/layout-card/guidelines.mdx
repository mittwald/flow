# Grundlagen

## Best practices

Achte bei der Verwendung einer LayoutCard darauf, dass...

- der Content innerhalb sinnvoll strukturiert ist. Dies kann beispielsweise durch die Components [Section](/03-components/structure/section) und [ColumnLayout](/03-components/structure/column-layout) erfolgen.
- sie standardmäßig in weiß eingefärbt ist. Wenn eine dekorative und farbige LayoutCard gewünscht ist, kann sie in Kombination mit der [AccentBox](/03-components/structure/accent-box) eingefärbt werden. 

## Verwendung

Verwende eine LayoutCard, um…

- einen oder mehrere Content-Bereiche auf einer Seite zu erzeugen.
- eine Kacheloptik auf einem Dashboard darzustellen.
- eine äußere Hülle für ein Navigationsmenü zu gestalten.
