# Playground

Verwende `<Accordion />`, um ein Accordion darzustellen. Nutze als Titel
entweder eine `<Heading />` oder ein `<Label />`.

# DefaultExpanded

`defaultExpanded` wird verwendet, um ein oder mehrere Accordion-Elemente beim
Laden der Seite standardmäßig geöffnet darzustellen.

# Outline Variant

Verwende eine `outline` Variante, wenn das Accordion klar vom umgebenden Inhalt
abgehoben oder inhaltlich stärker strukturiert werden soll.

# Mit Label

Statt einer [Heading](/03-components/content/heading/overview) kann auch ein
[Label](/03-components/content/label/overview) verwendet werden. Ein Label
eignet sich, wenn der Inhalt ergänzend oder sekundär ist und nicht im Fokus der
Seite steht – zum Beispiel bei Filtern, Formularfeldern oder optionalen
Einstellungen.
