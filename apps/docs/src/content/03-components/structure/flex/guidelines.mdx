# Grundlagen
Flex wurde eingeführt, um eine flexible Anordnung von Inhalten auf Basis 
der CSS-Flexbox zu ermöglichen. So bleibt unser Designsystem offen genug, 
um auch spezifische, schwer zu standardisierende Anforderungen und Layouts 
abzubilden. Dadurch ergänzen wir bestehende Layout-Muster, die durch 
[Align](/03-components/structure/align) abgebildet werden, um eine freiere 
Layout-Möglichkeit. 


## Best practices

Achte bei der Verwendung von Flex darauf, dass ...

- sie nicht für Layouts verwendet wird, die mit [Align](/03-components/structure/align) 
kombiniert werden können und einem festen UI-Pattern folgen.
- sie gezielt für Sonderfälle genutzt wird – in den meisten Fällen bietet unser 
Designsystem bereits eine passende Lösung.


## Verwendung

Verwende Flex, um …

- freie Layouts zu gestalten, die nicht durch bestehende Patterns abgedeckt werden.
- Components in einem Flex-Container gezielt zu zentrieren, auszurichten oder zu verteilen.
