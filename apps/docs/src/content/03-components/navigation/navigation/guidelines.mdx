# Grundlagen

## Best practices

Achte bei der Verwendung einer Navigation darauf, dass ...

- Menüeinträge kurz, prägnant und verständlich benannt sind.
- [Icons](/03-components/content/icon/overview) nur ergänzend und nicht als alleinige
  Information eingesetzt werden.
- maximal 7 bis 8 Menüeinträge in einer Gruppe enthalten sind, sodass die
  Navigation übersichtlich bleibt.
- einklappbare Gruppen den Überblick verbessern, jedoch die Orientierung
  erschweren können.

## Verwendung

Verwende eine Navigation, um Usern einen schnellen Zugriff auf Unterseiten zu
ermöglichen.
