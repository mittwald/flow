# Grundlagen
Die Navigation dient als zentrales Element zur Orientierung innerhalb einer Anwendung oder Website. Sie besteht aus mehreren Menüeinträgen, die zu unterschiedlichen Unterseiten führen. Um eine bessere Struktur bei vielen Menüpunkten zu ermöglichen, kann die Navigation in Gruppen unterteilt werden. 

## Best practices
Achte bei der Verwendung einer Navigation darauf, dass…
* Menüeinträge kurz, prägnant und verständlich benannt sind.
* [Icons](/03-components/content/icon) nur ergänzend und nicht als alleinige Information eingesetzt werden.
* Gruppen sinnvoll gebildet und thematisch klar voneinander abgegrenzt sind.
* maximal 7 bis 8 Menüeinträge in einer Gruppe enthalten sind, so dass die Navigation übersichtlich bleibt.
* einklappbare Gruppen den Überblick verbessern, jedoch die Orientierung erschweren können.


## Verwendung

Verwende eine Navigation, um Usern einen schnellen Zugriff auf Unterseiten zu ermöglichen.
