# Grundlagen

Der MessageThread gruppiert einzelne Nachrichten übersichtlich und macht so 
Gesprächsverläufe besser verständlich. Typischerweise wird er in einem 
[Chat](/03-components/chat/chat) eingesetzt


## Best Practices

Achte bei der Verwendung des MessageThread darauf, dass …

* die zeitliche Abfolge der Messages konsistent ist.
* wichtige Ereignisse (z. B. Statusänderungen oder der Eintritt neuer Teilnehmer) mittels
`<MessageSeparator />` hervorgehoben werden.

## Verwendung

Verwende den MessageThread, um ...

* einen Gesprächsverlauf mit mehreren [Messages](/03-components/chat/message) übersichtlich zu strukturieren.
* ihn im [Chat](/03-components/chat/chat) mit einer Texteingabe zu einem vollständigen Chat-Interface zu kombinieren.
