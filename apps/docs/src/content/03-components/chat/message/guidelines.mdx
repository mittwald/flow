# Grundlagen

## Best Practices

Achte bei der Verwendung der Message darauf, dass …

* mehrere Messages in einem [MessageThread](/03-components/chat/message-thread) gruppiert werden.
* der Inhalt der Message [Markdown](/03-components/content/markdown) unterstützt, damit Texteingaben korrekt formatiert dargestellt werden.
* das [ContextMenu](/03-components/actions/context-menu) nicht überladen wird und nur direkt relevante Aktionen (z. B. Bearbeiten und Löschen) enthält.

## Verwendung

Verwende die Message, um ...

* einzelne Nachrichten eines Chats darzustellen.
* in Kombination mit dem [MessageThread](/03-components/chat/message-thread) einen Gesprächsverlauf abzubilden.
