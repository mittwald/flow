# Grundlagen

## Best practices

Achte bei der Verwendung der Message darauf, dass ...

- mehrere Messages in einem [MessageThread](/03-components/chat/message-thread/overview)
  gruppiert werden.
- die [Markdown](/03-components/content/markdown/overview)-Component genutzt wird, wenn
  eine Texteingabe in Markdown zu erwarten ist.
- das [ContextMenu](/03-components/actions/context-menu/overview) nicht überladen wird
  und nur direkt relevante Aktionen (z. B. Bearbeiten und Löschen) enthält.

## Verwendung

Verwende die Message, um ...

- einzelne Nachrichten eines Chats darzustellen.
- in Kombination mit dem [MessageThread](/03-components/chat/message-thread/overview)
  einen Gesprächsverlauf abzubilden.
