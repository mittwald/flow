# Grundlagen

Die Chat-Component dient dem Aufbau eines vollständigen Chat-Interfaces. Sie kombiniert einzelne Components wie den [MessageThread](/03-components/chat/message-thread) und die Texteingabe durch die [TextArea](/03-components/form-controls/text-area) zu einer funktionalen Einheit. Dadurch entfällt für Entwickler die Notwendigkeit, diese Components manuell zusammenzustellen.

## Best Practices

Achte bei der Verwendung einer Chat-Komponente darauf, dass …

* das `aria-label` der Texteingabe bei abweichender Funktionalität angepasst wird.
* der Wert der `height` Property gut gewählt ist, um ausreichend Platz zum Lesen des Chats zu gewährleisten.

## Verwendung

Verwende die Chat-Component, um einen konsistenten und wiederverwendbaren Chatbereich in einer Anwendung bereitzustellen.
