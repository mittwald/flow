# Playground

Verwendet `<Chat />`, um einen Chat darzustellen.

Der Chat kombiniert dabei den [MessageThread](/03-components/chat/message-thread) mit einem Textinput, der mit Hilfe von [Align](/03-components/structure/align) einen [Button](/03-components/actions/button) und eine [TextArea](/03-components/form-controls/text-area) horizontal zu einander anordnet.  
Mit der `height` Property lässt sich die Höhe des Chats festlegen. Gleichzeitig sorgt sie dafür, dass die Texteingabe am unteren Rand platziert wird.
