# Playground

Verwende `<Chat />`, um einen Chat darzustellen.

Der Chat kombiniert dabei den
[MessageThread](/03-components/chat/message-thread) mit einem Textinput.
Letzterer ordnet mithilfe von [Align](/03-components/structure/align) einen
[Button](/03-components/actions/button) und eine
[TextArea](/03-components/form-controls/text-area) horizontal zueinander an.  
Mit der `height` Property lässt sich die Höhe des Chats festlegen. Gleichzeitig
sorgt sie dafür, dass die Texteingabe am unteren Rand platziert wird.

# Kombiniere mit ...

## Upload

Der Chat lässt sich durch einen Upload erweitern. Dazu kann ein
[Button](/03-components/actions/button) mit einem
[Icon](/03-components/content/icon) in der Variante **Plain** ergänzt werden.
Die hochgeladenen Files lassen sich mit der
[FileCard](/03-components/upload/file-card) darstellen.
