# Indeterminated

Die Property `isIndeterminate` zeigt an, dass sich eine Checkbox in einem visuell uneindeutigen Zustand befindet – also weder vollständig ausgewählt noch vollständig abgewählt ist. Sie wird verwendet, um übergeordnete Checkboxen darzustellen, bei denen nur ein Teil der untergeordneten Optionen ausgewählt wurde.

# Disabled

In bestimmten Fällen kann eine Checkbox den State **Disabled** haben. In diesem Fall ist die Auswahloption für den User deaktiviert und kann nicht ausgewählt werden.
