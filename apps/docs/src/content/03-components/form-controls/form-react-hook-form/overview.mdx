# Aktionen nach Submit ausführen

Aktionen die _nach_ dem Submit ausgeführt werden sollen, wie z.B. ein Form-Reset
oder einen Redirect, sollten im Return-Callback des Submit Handler ausgeführt
werden. Dieser kann auch asynchron sein.

```
const submitHandler = () => {
  // submit logic
  return () => {
    form.reset();
  };
};
```
