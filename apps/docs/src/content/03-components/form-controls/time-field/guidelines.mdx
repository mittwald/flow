# Grundlagen

## Best practices

Achte bei der Verwendung eines TimeFields darauf, dass…

- ein verständliches [Label](/03-components/content/label) wie „Uhrzeit“ verwendet wird.
- das Zeitformat dem vertrauten Format des Users und dem Nutzungskontext entspricht.
- es typischerweise in Formularen eingesetzt wird.

## Verwendung

Verwende ein TimeField, um dem User eine konkrete Uhrzeit auswählen zu lassen – z. B. für einen Termin.
