mitgegeben werden. Der Default-Wert kann über das Property `showInitialMarker`
auf dem Slider markiert werden. Außerdem werden diverse Formate und Einheiten
unterstützt (s.
[Intel.NumberFormat](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/NumberFormat)
