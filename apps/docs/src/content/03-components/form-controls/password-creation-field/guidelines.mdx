# Grundlagen

Das PasswordCreationField ist eine Component zur Unterstützung bei der
Erstellung sicherer Passwörter. Es bietet eine visuelle Anzeige der
Passwortkomplexität und erleichtert es Usern, unsichere Passwörter zu erkennen
und zu vermeiden. Standardmäßig werden vordefinierte Passwortregeln angewendet,
alternativ können eigene Richtlinien konfiguriert werden. Für die Eingabe
bestehender Passwörter sollte hingegen ein reguläres
[TextField](/03-components/form-controls/text-field/overview) mit dem Type `password`
verwendet werden.

## Best practices

Achte bei der Verwendung eines PasswordCreationFields darauf, dass ...

- standardisierte Richtlinien genutzt werden, sofern keine spezifischen
  Anforderungen bestehen.
- selbst formulierte Richtlinien klar und eindeutig für die User verständlich
  sind.

## Verwendung

Verwende ein PasswordCreationField, um ...

- Usern bei der sicheren Erstellung neuer Passwörter zu unterstützen.
- visuelles Feedback zur Stärke und Gültigkeit eines Passworts bereitzustellen.
- die Generierung sicherer Passwörter innerhalb definierter Richtlinien
  anzubieten.
- Passwortanforderungen transparent und nachvollziehbar zu kommunizieren.
