# Playground 

Verwende `<PasswordCreationField />`, um ein PasswordCreationField dazustellen. Standardmäßig enthält die Komponente folgende Passwortregeln:
* Mindestlänge von 12 Zeichen
* Mindestens ein Sonderzeichen enthalten
* Mindestens eine Zahl enthalten
* Darf nicht durch HaveIBeenPwned verboten sein
