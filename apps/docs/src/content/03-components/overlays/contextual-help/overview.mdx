# Playground
Verwende <ContextualHelp />, um ein ContextualHelp darzustellen. Das ContextualHelp kann aus [Heading](/03-components/content/heading), [Text](/03-components/content/text) und [Link](/03-components/navigation/link) bestehen.

Da das ContextualHelp sich nur durch eine Aktion des Users anzeigen lässt, wird ein <ContextualHelpTrigger /> in Kombination mit einem <Button /> benötigt.
