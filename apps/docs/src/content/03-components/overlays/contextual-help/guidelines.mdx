# Grundlagen

## Best practices
Achte bei der Verwendung des ContextualHelp darauf, dass …

- es nicht alleinstehend verwendet wird, sondern immer einem spezifischen Element (z. B. [Heading](/03-components/content/heading),
[Text](/03-components/content/text) oder [Label](/03-components/content/label)) zugeordnet wird.
- es nicht für Inhalte benutzt wird, die zum Verständnis der Benutzeroberfläche unerlässlich sind.
- es stattdessen für weiterführende Erklärungen oder zum Verweisen auf externe Quellen verwendet wird.

## Verwendung

Verwende ein ContextualHelp, um...

- Inhalte oder Forms in einem [Modal](/03-components/overlays/modal) mit weiterführenden Informationen anzureichern, ohne das Modal optisch zu überladen.
- spezifische Daten oder Inhalte in einem Dashboard näher zu erläutern.
- auf weiterführende Quellen zu inhaltlich komplexeren Themen zu verweisen.
