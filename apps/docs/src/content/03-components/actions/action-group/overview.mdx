# Slots

Die Actions werden von der ActionGroup automatisch in die Slots `primary`, `secondary` und `abort` sortiert. So werden Buttons mit der Color Primary (Standard), Accent und Danger automatisch im Slot `primary` der an Ende der ActionGroup sortiert.
