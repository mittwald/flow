# Grundlagen

Die ActionGroup dient dazu, mehrere Actions sinnvoll zu gruppieren und in eine
konsistente Reihenfolge zu bringen. Sie wird häufig im
[Section](https://mittwald.github.io/flow/03-components/structure/section)
Header, im Header einer
[List](https://mittwald.github.io/flow/03-components/structure/list) oder im
Footer eines
[Modals](https://mittwald.github.io/flow/03-components/overlays/modal)
verwendet. Sie kann aber auch eigenständig im Content eingesetzt werden.

## Best practices

Achte bei der Verwendung einer ActionGroup darauf, dass ...

- die innerhalb der ActionGroup eingesetzten Buttons die Richtlinien aus der
  [Button Guideline](https://mittwald.github.io/flow/03-components/actions/button)
  einhalten.
- bei mehreren Buttons nur ein Button als Hauptaktion hervorgehoben wird.
- sie häufig in anderen Components wie
  [Section](https://mittwald.github.io/flow/03-components/structure/section),
  [List](https://mittwald.github.io/flow/03-components/structure/list) oder
  [Modal](https://mittwald.github.io/flow/03-components/overlays/modal) zum
  Einsatz kommt.

## Verwendung

Verwende eine ActionGroup, um ...

- mehrere zusammengehörige Actions in eine sinnvolle Reihenfolge zu bringen.
- eine Hauptaktion der Section, des Modals oder der Liste hervorzuheben.
