Für das Flow Design System werden [Tabler Icons](https://tabler.io/icons) verwendet. Unten in der Icon-Bibliothek ist eine Auflistung aller Icons zu finden, denen im Flow Design System eine Bedeutung zugeordnet ist.

Um ein Icon einzufügen, muss innerhalb eines Tags „Icon“ gefolgt vom jeweiligen Icon-Namen aus der Auflistung verwendet werden. Beispiel: Für das Icon „Home“ sieht der Tag so aus: `<IconHome />`.

# Icon-Bibliothek

Im Flow Design System haben ausgewählte Icons eine festgelegte Bedeutung. Sie lassen sich drei Typen zuordnen: **Functional, Decorative** und **Status**.

## Functional

Functional Icons besitzen eine eindeutige, allgemein verständliche Symbolik und müssen nicht neu erlernt werden. Sie unterstützen User bei Navigation und Interaktion und werden häufig in Actions wie [Icon-Buttons](/03-components/actions/button) eingesetzt.

<LiveCodeEditor example="functional" editorCollapsed />

## Decorative

Decorative Icons tragen zur Ästhetik und zur emotionalen Wirkung der Benutzeroberfläche bei. Im Gegensatz zu den Functional Icons ist ihre Zuordnung jedoch oft nicht intuitiv erfassbar.
