# Level

Über das Property `level` kann die Hierarchie der Überschrift angepasst werden.
Standardmäßig wird eine Heading mit dem Level `H2` erzeugt. Die Seite sollte
eine sinnvolle Struktur mit verschiedenen Ebenen aufweisen.

<LiveCodeEditor example="levels" editorCollapsed />

---

# Size

Über das Property `size` kann die Schriftgröße einer Heading manuell angepasst
werden. Die Überschriften sollten dabei eine visuelle Hierarchie beibehalten.

# Color

Die Heading wird standardmäßig mit der Primärfarbe `primary` dargestellt.

## Color Light und Dark

Zusätzlich zur Standardfarbe kann die Heading in den Varianten Light und Dark
dargestellt werden. Diese Farbvarianten eignen sich, wenn die Standardfarbe auf
farbigen oder dekorativen Hintergründen nicht gut funktioniert.

<LiveCodeEditor example="dark" bgColor="light" editorCollapsed />

<LiveCodeEditor example="light" bgColor="dark" editorCollapsed />

**Light:** Verwende die Light-Variante bei dunklen Hintergründen mit einem
HSL-Lightness-Wert unter 50.

**Dark:** Verwende die Dark-Variante bei hellen, farbigen Hintergründen mit
einem HSL-Lightness-Wert über 50.
