# Grundlagen

## Best practices

Achte bei der Verwendung von Text darauf, dass ...

- in Navigationen, bei denen der aktive Menüpunkt `bold` ist, mit `emulateBoldWidth` gearbeitet wird.
- Absätze über `elementType` korrekt als `<p>` gekennzeichnet werden.

## Verwendung

Verwende Text, um …

- Fließtext in der Benutzeroberfläche darzustellen.
- den Text innerhalb einer Component zu bearbeiten.
