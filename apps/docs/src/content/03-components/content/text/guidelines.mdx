# Grundlagen

## Best practices

Achte bei der Verwendung von Text darauf, dass ...

- Texte prägnant, verständlich und hilfreich sein sollen. Beachte zum
  konsistenten Umgang mit Text auch den
  [Sprach-Guide](/02-foundations/03-content-guidelines/01-sprach-guide).
- Text häufig nicht oder nur unaufmerksam wahrgenommen wird, sodass eine
  Benutzeroberfläche auch ohne zusätzliche Erläuterungen funktionieren sollte.
- die Zeilenlänge 75 Zeichen nicht überschreitet. Längere Zeilen erschweren die
  Lesbarkeit und das Verständnis des Inhalts.
- Absätze über `elementType` korrekt als `<p>` gekennzeichnet werden.

## Verwendung

Verwende Text, um ...

- Fließtext in der Benutzeroberfläche anzuzeigen.
- Text innerhalb von Components darzustellen.
