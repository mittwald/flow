# Color

Die Color bei der Kombiniation mit [Icon](/03-components/content/icon) oder Initialen richtet sich jeweils nach dem übergebenen Child der Componente. Für eine gezielte Wahl der Color kann das Property `color` verwendet werden.

Folgende Colors stehen zur Verfügung: **Blue, Violet, Teal, Lilac** und **Green**.

<LiveCodeEditor example="colors" editorCollapsed />

---

# Status

Wird im Avatar das Property `status` gesetzt erhält der Avatar automatisch das
passende Icon und die passende Color. Dieses Property darf nur verwendet werden, wenn der Status zusätzlich durch ein benachbartes Element (z. B. [Text](/03-components/content/text), [Label](/03-components/content/label) oder [Legende](/03-components/content/legend)) eindeutig kommuniziert wird.
