# Farbvarianten

Die Farbe hängt von den jeweiligen Children der Komponente ab, kann aber über
das Property `color` überschrieben werden um eine bestimmte Farbvariante zu
erhalten. Sie dient dazu für Abwechslung in den angezeigten Avataren zu sorgen.

<LiveCodeEditor example="colors" editorCollapsed />

# Status

Wird im Avatar das `status` Property gesetzt erhält der Avatar automatisch das
passende Icon und die passende Farbe. Das Property darf nur verwendet werden,
wenn der Status durch ein nahe gelegenes Element (z.B. Text, Label oder Legende)
deutlich gemacht wird.
