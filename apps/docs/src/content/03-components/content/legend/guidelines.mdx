# Grundlagen

Die Legend ordnet visuell Namen oder Kategorien zu verschiedene [Categorical Colors](/02-foundations/01-design/02-colors) zu, sodass User Elemente innerhalb von z. B. Diagrammen zuordnen können. Die verwandte Component ChartLegend hat die gleichen Eigenschaften einer Chart, mit dem Unterschied, dasss diese speziell für die CartesianChart gedacht ist.

## Best practices

Achte bei der Verwendung einer Legend darauf, dass...

- die Farbzuordnung exakt den Categorical Colors der dargestellten Elemente entspricht, um eine eindeutige Wiedererkennung zu gewährleisten. Die Categorical Colors sollen einen klaren visuellen Unterschied zwischen den Elementen ermöglichen, um die Erfassung zu erleichtern.
- sie in der Nähe der zugeordneten Elemente platziert wird, sodass User auf einen Blick die Bedeutung der Farben erfassen können.
- pro Diagramm nur eine Legend mit mehreren LegendItems verwendet wird.
- die Reihenfolge der LegendItems logisch ist und idealerweise der Anordnung im Diagramm entspricht.

## Verwendung

Verwende eine Legend, um Usern die Bedeutung farbiger Elemente, z. B. in Diagrammen, verständlich zu machen.
