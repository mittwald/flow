# Grundlagen

InlineCode hebt Code oder codeähnliche Begriffe im Fließtext hervor. Dabei
handelt es sich meist um Funktionsnamen, Variablen, Dateinamen oder Pfade.

## Best practices

Achte bei der Verwendung von InlineCode darauf, dass ...

- er sparsam eingesetzt wird. Zu viele InlineCodes können einen Text unleserlich
  machen.
- er nur für kurze Beispiele verwendet wird. Bei längeren Code-Beispielen
  empfiehlt sich der [CodeBlock](/03-components/content/code-block/overview).

## Verwendung

Verwende InlineCode, um ...

- einzelne Code-Fragmente im Fließtext darzustellen.
- Usern auf den ersten Blick zu zeigen: Hier handelt es sich um Code.
