# Grundlagen

InlineCode hebt Code oder Code-ähnliche Begriffe im Fließtext hervor.
Meist handelt es sich dabei um Funktionsnamen, Variablen, Dateinamen oder Pfade. 

## Best practices

Achte bei der Verwendung von InlineCode darauf, dass…

- er sparsam verwendet wird. Zu viele InlineCodes können ein Text unleserlich werden lassen.
- er nur für kurze Beispiele verwendet wird. Bei längeren Code-Beispielen empfiehlt sich der [CodeBlock](/03-components/content/code-block).

## Verwendung

Verwende InlineCode, um …

- einzelne Code-Fragmente im Fließtext darzustellen.
- Usern auf den ersten Blick zu zeigen: Hier handelt es sich um Code.
