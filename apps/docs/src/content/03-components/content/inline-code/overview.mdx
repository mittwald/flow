# Color

Zusätzlich zur Standard-Color kann InlineCode in den Colors **Light** und
**Dark** dargestellt werden. Nutze diese Varianten, wenn die Standard-Color auf
farbigen oder dekorativen Hintergründen keinen ausreichenden Kontrast (mind.
4,5:1) bietet.
