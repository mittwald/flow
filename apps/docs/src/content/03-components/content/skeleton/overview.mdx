# Accessibility

Das Skeleton verwendet das Property `aria-hidden`, damit es vom Screenreader ignoriert wird. 

Die Accessibility eines Ladezustands soll **nicht** vom Skeleton geregelt werden.

Um Screenreadern mitzuteilen, dass Inhalte noch geladen werden, sollte das Property `aria-busy` verwendet werden. Dieses wird auf `false` gesetzt, sobald der Ladevorgang abgeschlossen ist (s.
[MDN](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-busy)).
