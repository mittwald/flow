# Grundlagen

## Best practices

Achte bei der Verwendung vom CodeBlock darauf, dass …

- er nur für Beispiele genutzt wird, die länger als eine Zeile sind.
- bei großen Beispielen Zeilennummern verwendet werden, um die Orientierung zu verbessern.
- das Code-Beispiel nur dann kopierbar ist, wenn es sinnvoll an anderer Stelle weiterverwendet werden kann.

## Verwendung

Verwende den CodeBlock, um …

- Code-Beispiele übersichtlich und mit dem richtigen Syntax-Highlighting darzustellen.
- dem User eine einfache Möglichkeit zu bieten, Code-Snippets zu kopieren.
