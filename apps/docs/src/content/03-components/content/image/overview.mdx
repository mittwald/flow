# Responsive Image Resolution

Um unterschiedliche Bild-Auflösungen für diverse Bildschirmgrößen anzubieten
eignet sich das `srcSet` Attribut. Für diverse Pixeldichten auf Nutzerseite
können hiermit verschiedene niedrig- oder hochauflösende Varianten der
Quelldatei hinterlegt werden. Näheres hierzu ist in folgendem Artikel einsehbar:
[Responsive Images](https://developer.mozilla.org/en-US/docs/Learn/HTML/Multimedia_and_embedding/Responsive_images#resolution_switching_same_size_different_resolutions:~:text=Resolution%20switching%3A%20Same%20size%2C%20different%20resolutions).
