# Grundlagen

Die Rating Component dient zur visuellen Darstellung einer Bewertungsskala von 0 bis 5 Sternen. Sie wird häufig eingesetzt, um Bewertungen auf einen Blick verständlich zu machen. Typische Anwendungsfälle sind Userfeedback sowie die Bewertung von Produkten und Leistungen. 

## Best practices

Achte bei der Verwendung eines Ratings darauf, dass…
* die visuelle Darstellung immer durch ergänzenden Kontext unterstützt wird, z. B. durch eine Beschriftung oder eine Kombination mit anderen Components wie der [BigNumber](https://mittwald.github.io/flow/03-components/data-visualisation/big-number).
* eine konsistente Verwendung innerhalb eines Interfaces gewährleistet ist, etwa durch eine einheitliche Skalierung (0 bis 5 Sterne).
* unterstützende visuelle Elemente (z.B. der [AccentBox](https://mittwald.github.io/flow/03-components/structure/accent-box)) eingesetzt werden können, um das Rating optisch hervorzuheben.

## Verwendung

Verwende eine Rating, um…
* Userbewertungen sichtbar zu machen, z. B. bei Rezensionen oder Produktempfehlungen.
* den Kontext einer Zahl oder Kennzahl (z. B. der [BigNumber](https://mittwald.github.io/flow/03-components/data-visualisation/big-number)) zu verdeutlichen, etwa zur Einordnung auf einer Bewertungsskala.
* Einschätzungen oder qualitative Urteile schnell erfassbar darzustellen, etwa bei Auswertungen.
