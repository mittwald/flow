# Grundlagen

Die Rating Component dient zur visuellen Darstellung einer Bewertungsskala von 0 bis 5 Sternen. Sie wird häufig eingesetzt, um Bewertungen auf einen Blick verständlich zu machen. Typische Anwendungsfälle sind Userfeedback oder zur Bewertung von Produkten und Leistungen. 

# Best Practices

Achte bei der Verwendung einer Rating darauf, dass…
* die visuelle Darstellung immer durch ergänzenden Kontext unterstützt wird, z. B. durch eine Beschriftung oder eine Kombination mit anderen Components wie der [BigNumber](https://mittwald.github.io/flow/03-components/data-visualisation/big-number).
* eine konsistente Verwendung innerhalb eines Interfaces, z. B. immer die gleiche Skalierung (0 bis 5 Sterne)
* den Einsatz unterstützender visueller Elemente (z.B. der [AccentBox](https://mittwald.github.io/flow/03-components/structure/accent-box)), um sie visuell hervorzuheben.

# Verwendung

Verwende eine Rating, um…
* Userbewertungen sichtbar zu machen, z. B. bei Rezensionen oder Produktempfehlungen.
* den Kontext einer Zahl oder Kennzahl (z.B. der [BigNumber](https://mittwald.github.io/flow/03-components/data-visualisation/big-number)) zu verdeutlichen, etwa zur Einordnung auf einer Bewertungsskala.
* Einschätzungen oder qualitative Urteile schnell erfassbar darzustellen, etwa bei internen Scores oder Auswertungen.
