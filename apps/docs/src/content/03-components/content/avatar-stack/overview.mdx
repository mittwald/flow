# Mit Total Count

Wenn die Liste an Avataren so lang ist, dass du nicht alle anzeigen möchtest
(wir empfehlen höchstens 5 Avatare anzuzeigen), kannst du das `totalCount`
Property verwenden. Sobald der totalCount größer ist als die Anzahl an Avataren,
wird ein zusätzliches Element mit der übrigen Anzahl angezeigt
