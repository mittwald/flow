# Mit EmptyView

Der EmptyView wird angezeigt, sobald die `data` des Charts entweder ein leeres
Array enthält, oder komplett `undefined` ist. Somit kann sie für
Ausnahmezustände wie Ladefehler, Ladezustände oder leere Datenergebnisse genutzt
werden.
