# Grundlagen

## Best Practices

Achte bei der Verwendung eines CartesianCharts darauf, dass…
- die x-Achse immer zeitbasiert ist.
- Dimensionen mit [Categorical Colors](Link) dargestellt werden.
- bei der Area-Darstellung die Flächen nach Größe sortiert werden, mit der größten Fläche unten.
- die Anzahl der dargestellten Dimensionen übersichtlich bleibt, um Lesbarkeit sicherzustellen.
- ein LineChart gewählt wird, wenn die Fläche keinen zusätzlichen Mehrwert liefert.

## Verwendung

Verwende eine CartesianChart, um…
- Trends von Werten über längere Zeiträume hinweg zu zeigen.
- den relativen Anteil mehrerer Gruppen an einem Gesamtwert darzustellen (mit Area-Darstellung).
- Veränderungen bei einer oder mehreren Dimensionen miteinander zu vergleichen.
