# Section-Inhalte bearbeiten

Durch die generelle Strukturierung von Inhalten mit einer
[Section](/03-components/structure/section/overview) ist es möglich, im Header der
Section einen „Bearbeiten“-[Button](/03-components/actions/button/overview) zu
platzieren. Dieser öffnet je nach Länge des Bearbeitungsformulars ein
Standard-[Modal](/03-components/overlays/modal/overview) oder ein Modal in der
OffCanvas-Variante.
