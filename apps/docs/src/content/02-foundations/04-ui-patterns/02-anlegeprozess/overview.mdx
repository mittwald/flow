# List

## Leerer Inhalt

Wenn sich noch kein Element in der [List](/03-components/structure/list/overview)
befindet, sollte statt der List eine
[IllustratedMessage](/03-components/content/illustrated-message/overview) verwendet
werden. Ein Klick auf den [Button](/03-components/actions/button/overview) öffnet in den
meisten Fällen ein [Modal](/03-components/overlays/modal/overview) – bei einfachen
Anlegeprozessen in der üblichen Modal-Darstellung, bei umfangreicheren Prozessen
in der OffCanvas-Variant.

## Mit Inhalt

In einer List kann oben rechts eine
[ActionGroup](/03-components/actions/action-group/overview) platziert werden, die einen
Button enthält, über den weitere Elemente angelegt/gebucht werden. Auch hier
öffnet ein Klick auf den Button entweder ein Anlege-Modal für kleinere Prozesse
oder ein Modal in der OffCanvas-Variante für größere Anlege-/Buchungsprozesse.
